How it works

To obtain the data and images, Posterfy uses Spotify's free API. After the search result and the user selects the album, Posterfy gathers all the data and organizes it visually using the HTML canvas element in a pre-organized way through JavaScript and generates a rendered image of the canvas.

This project is open source and does not generate any revenue. All rights to images and information belong to Spotify and the respective artists. Source code available at: Github
